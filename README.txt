# Installation Instructions

## Altium 18
Within your project open the libraries tab from the bottom right "Panels" > "Libraries",
Click "Libraries..."
Click "Install" > "Install From File"
Select the compiled library from the project output folder of this repository.

## Altium 19
Within your project open the components tab from the bottom right "Panels" > "Components"
click on the button with 3 horzontal lines at the top of the panel
Click "File-based Libraries Preferences..."
Select the "Installed" tab at the top
Click "Install..." at the bottom right
Select the compiled library from the project output folder of this repository.
